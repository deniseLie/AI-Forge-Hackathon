# 60's Pulse — Pitch Deck Outline

**Narrative spine:** A scandal front page that hasn't happened yet → the problem it represents →
our panel of 60 grounded agents → the Blast Score + the second that detonated → the 3-tier fix that
separates rewrite-able from re-shoot from no-go → the sponsors that make it real → the ask.

Every slide is a single 720×405 broadsheet frame. The deck *is* the dashboard: Fraunces black
headlines, IBM Plex Mono for every number, blackletter only for the nameplate, newsprint-on-ink
(`--ink #0B0C10`, `--paper #F3EFE3`), heat as fill never as red text.

## Slide 2 — The problem

## Slide 3 — The solution
**Type:** One-line value prop + what you get back
**Key message:** 60 grounded Singaporean agents, each sandboxed, give you a Blast Score before launch.

- Headline: **Ask Singapore first.**
- The four deliverables as a broadsheet index: a **Blast Score %** · a **timeline blast map**
  (which second detonated) · **tomorrow's fictional front page** · a **3-tier fix triage**
- One line on what's different: not sentiment scoring — 60 *characters*, self-grounded in this
  week's discourse, each isolated in its own sandbox.

## Slide 4 — How it works (architecture)
**Type:** Pipeline diagram (tldraw asset)
**Key message:** Untrusted public text enters only inside per-agent sandboxes; the orchestrator stays clean.

- Left-to-right flow: **campaign in** → **VideoDB** ingest (scenes/keyframes/transcript) →
  **Kimi K2.6 orchestrator** (trusted host, plans + dispatches) → **60 Daytona sandboxes**
  (each: Bright Data self-ground + Kimi multimodal react) → **AGGREGATE** → dashboard
- Callout: each agent self-grounds in untrusted public text (an injection vector that shapes its
  *identity*) → Daytona contains the blast radius per agent. The orchestrator only touches structured
  manifests, so the isolation boundary maps exactly to where untrusted input enters.
- Honest caveat chip: baked once offline (60 cold-starts is a stage killer), replayed live.

## Slide 6 — The 3-tier fix triage
**Type:** Step-down chart + three tier cards
**Key message:** We separate what a rewrite saves from what needs a re-shoot from what's simply a no-go.

- Step-down: **84 → 55 → 28** (residual 28 stamped "DECISION · your call")
- Three cards: **COPY** rewrite the privacy line (84→55) · **PRODUCTION** cut the 0:12 scene,

- The Goodhart kill: "Couldn't a rewrite game your panel?" — wording only moves the wording tier, by construction.
